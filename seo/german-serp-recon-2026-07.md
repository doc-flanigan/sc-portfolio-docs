## Terminology (community register — use in all /de/ copy; du-form ALWAYS)

- Game package = "Spielpaket" / untranslated "Game Package"; starter = **"Starterpaket"**, ship = "Starterschiff"
- Pledge/Warbond/Store Credits/Free Fly/Wipe/UEC/aUEC = untranslated (der Wipe, "pledgen")
- Referral = dual register: **"Empfehlungscode"** AND "Referral-Code" (never "Recruiting Code")
- Bonus = **"50.000 UEC Startguthaben"** (killer native phrase)
- Real money = **"Echtgeld"**; counter-phrase "alle Schiffe können im Spiel erspielt werden"
- Intent phrases: "kostenlos spielen/testen", "lohnt sich", world = "das Verse"
- Insurance = "Versicherung"/"Lebenslange Versicherung" (LTI)
