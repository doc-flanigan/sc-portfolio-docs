2026-07-14: frontmatter status was wrongly `refuted` (bad write on 2026-07-12); corrected to `verified` during the first dev-post cross-audit, which also surfaced the strongest official source yet — the 2023-10-11 Spectrum staff post "Star Citizen: Community Localization Update" (now in sources): CIG states official localization is in progress long-term and explicitly blesses community global.ini translation files in the meantime. That is precisely this claim.

2012 stretch-goal comm-link 12760 promised EFIGS ('English, French, Spanish and German') localization 'as time goes on' but this is a 14-year-old pre-production aspiration, never confirmed delivered. Alpha 3.24.0 patch notes (comm-link 20087, Sep 2024 — the version cited by fan-site copy as when German landed) contain no language/localization feature entry. Comm-Link full-text search for 'German localization', 'language support', and 'subtitles' returns zero hits across the entire archive. Correct copy: game client is English-only officially; German requires unofficial global.ini community translation files.
